## はじめに — なぜこの2つなの？

小学生に持たせる携帯は、大きく分けて **2つのタイプ** があります。

| タイプ | 代表的な製品 | 特徴 |
|-------|-----------|------|
| **キャリアのキッズケータイ** | ドコモ、au、ソフトバンク | 電話回線で通話できる。親と同じキャリアが基本 |
| **SIMフリーのキッズスマホ** | Hamic MIELS nico | ネット回線で通話。親のキャリアは何でもOK |

日本で買える「キッズ向け端末」はこの2タイプしかありません。

この記事では、**キャリア型の代表「ドコモ キッズケータイ KY-41C」** と、**SIMフリー型で唯一の選択肢「Hamic MIELS nico」** を徹底比較します。

## GPS見守り機能の比較

> **「子供が今どこにいるか」を知るための機能。一番大事なポイントです。**

### ドコモ キッズケータイの場合

- **イマドコサーチ**（月額330円）で親のスマホから子供の位置を確認
- 子供が設定したエリアを出入りすると通知（みまもりアラート）
- **弱点**: 電源がOFFだと追跡できない。屋内では精度が落ちる

```
【ドコモのGPS】

  親のスマホ ←──── GPS衛星 ────→ キッズケータイ
     📱                              📞

  ※ 電源が入っているときだけ追跡可能
  ※ 屋内・地下では精度が落ちる
```

### Hamic MIELS nico の場合

- **独自のビーコン技術**で、電源が切れても位置情報を発信し続ける
- 全国 **80,000箇所の見守りスポット** がビーコン信号をキャッチ
- 子供が移動中は **1分間隔** で位置情報を更新
- 止まっているときはバッテリー節約のため間隔を広げる（賢い！）
- エリアの出入り通知（ジオフェンス）にも対応

```
【HamicのGPS】

  親のスマホ ←── GPS衛星 ──→ Hamic端末
     📱                         📱
                                 │
                    電源OFFでも！ │
                                 ▼
                          ビーコン発信
                                 │
              見守りスポット ─────┘
             （全国80,000箇所）
```

### GPS見守りの結論

| | ドコモ | Hamic |
|--|:------:|:-----:|
| 電源ON + 屋外 | ○ | ◎ |
| 電源ON + 屋内 | △ | ◎ |
| **電源OFF** | **✕ 追跡不能** | **○ ビーコンで追跡可能** |

> **Hamic の圧勝です。** 特に「電源が切れても追跡できる」のは、子供の安全を考えると非常に大きなメリットです。万が一、子供が端末の電源を切ってしまったり、バッテリーが切れても、位置がわかります。

## 通話機能の比較

> **「子供が誰かに電話できるか」— ここが一番の分かれ道です。**

### ドコモ キッズケータイの場合

- 普通の **電話回線** で通話（090/080番号）
- 登録した相手にだけ電話できる（知らない番号には発信できない設定が可能）
- **110番（警察）・119番（消防）にかけられる**
- おじいちゃん・おばあちゃんの携帯にも、普通に電話できる
- +メッセージ（ショートメール的なもの）も使える

```
【ドコモの通話】

  子供 ──→ 電話回線（090/080）──→ おじいちゃんの携帯
  📞                                    📱

  ※ 相手は何もインストールする必要なし
  ※ 110番・119番もOK
```

### Hamic MIELS nico の場合

- **Hamicアプリ** を使ったネット通話のみ
- 電話番号での発信・着信は **一切できない**
- 110番・119番にも **かけられない**
- 相手もHamicアプリをインストールする必要がある
- Wi-Fiまたはモバイル通信（月1GB）が必要

```
【Hamicの通話】

  子供 ──→ インターネット ──→ Hamicアプリ ──→ 親のスマホ
  📱          (Wi-Fi/4G)        📲              📱

  ※ 相手もHamicアプリのインストールが必要
  ※ 電話番号では通話できない
  ※ 110番・119番もかけられない
```

### 通話の結論

| | ドコモ | Hamic |
|--|:------:|:-----:|
| おじいちゃんに電話 | ◎ 普通にかけられる | △ アプリ必要 |
| 110番・119番 | ◎ かけられる | ✕ かけられない |
| 通話の安定性 | ◎ 電話回線で安定 | ○ ネット依存 |
| 通話料 | 家族間無料、他は有料 | 無料（アプリ同士） |

> **ドコモの圧勝です。** 特に緊急時に110番・119番に電話できるかどうかは、子供の安全に直結します。おじいちゃん・おばあちゃんなど、アプリを入れるのが難しい相手にも普通に電話できるのは大きなメリットです。

## 防犯機能の比較

> **いざというとき、子供を守る機能。**

### ドコモ キッズケータイ

- 防犯ブザー（約100dBの大音量 — 電車の通過音くらい）
- ブザー作動時に防犯ライトが点滅
- ブザーを引くと自動的に保護者に通知
- みまもりアラート（エリア離脱通知）

### Hamic MIELS nico

- 防犯ブザー内蔵（端末一体型）
- ブザー作動時に **周囲の音を自動録音して保護者に送信**
- ブザー作動時に **現在地を自動送信**
- ジオフェンス（エリア出入り通知）

### 防犯機能の結論

| 機能 | ドコモ | Hamic |
|------|:------:|:-----:|
| 防犯ブザー | ○ | ○ |
| 大音量 | ◎（100dB） | ○ |
| 防犯ライト | ○ あり | ✕ なし |
| **ブザー時の自動録音** | **✕ なし** | **◎ あり** |
| ブザー時の位置送信 | ○ あり | ◎ あり |
| エリア離脱通知 | ○ あり | ○ あり |

> **僅差でHamicが上。** 「自動録音」機能は、万が一のとき周囲の音声が記録されるため、証拠になります。一方、ドコモの防犯ライト（ピカピカ光る）も、周囲への注意喚起としては効果的です。

## よくある質問

### Q. 親がpovo（格安SIM）でもドコモのキッズケータイは使える？

**A. GPS見守り（イマドコサーチ）は使えます。** 2023年12月の改定で、ドコモ以外のキャリアでもイマドコサーチが利用可能になりました。ただし、家族間通話無料は適用されません。

### Q. Hamicで110番にかけられないのは危なくないの？

**A. 確かに弱点です。** ただし、防犯ブザーを引けば自動的に保護者に通知＋位置情報が送信されるので、親が代わりに110番することは可能です。とはいえ、子供が自分で直接110番できるドコモの方が、緊急時の安全性は高いと言えます。

### Q. おじいちゃんにHamicアプリを入れてもらうのは現実的？

**A. スマホに慣れているおじいちゃんなら大丈夫です。** ただし、ガラケーを使っているおじいちゃんや、アプリのインストールが難しいおじいちゃんには厳しいです。電話番号でかけられるドコモの方が確実です。

### Q. 両方持たせるのはアリ？

**A. 小学1年生に2台は現実的ではありません。** どちらか1つに絞ることをおすすめします。

### Q. LINEは使える？

**A. どちらも使えません。** ドコモのキッズケータイにはLINEアプリが入りません。HamicはSMS認証ができないためLINEを使えません。どちらもキッズ向けの安全な設計です。

## まとめ

### 一番大事なポイント

この2つの端末は **「何を一番大事にするか」** で選び方が変わります。

| あなたが一番大事にすること | おすすめ |
|-------------------------|---------|
| **電話回線での通話** | → ドコモ キッズケータイ |
| **GPS見守りの精度** | → Hamic MIELS nico |

### 筆者の見解

**どちらも「子供を守る」という目的では優秀な端末です。**

- **電話回線（110番含む）は、いざというときの安全の「最後の砦」** です。おじいちゃん・おばあちゃんへの連絡も含め、電話回線の安心感は大きい。
- **一方で、GPSの見守り精度ではHamicが圧倒的に優れています。** 電源が切れても追跡できるのは、他のどのキッズケータイにもない機能です。

ご家庭の状況に合わせて、最適な1台を選んでください。
